# 🛍️ Shoeshop Server

A nodejs server for e-commerce application called ShoeShop

### Setup Steps

1. Clone repository:

```bash
git clone https://github.com/ptduy2603/nodejs-shoeshop-server.git
```

2. Install dependencies

```bash
npm install
```

3. Start project on development environment

```bash
npm start
```

## 💻 Tech stack

- NodeJS (ExpressJS)
- Mongoose
- MVC model
